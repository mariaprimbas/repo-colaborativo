# repo-colaborativo

Integrantes del grupo: Catalina Abadia, Gloria Alderete Cornejo, Olivia Kemmerer y Maria Primbas 

El repositorio contiene los archivos que permiten utilizar el programa que registra y analiza los datos ingresados por el usuario sobre sus habitos. Para ello, se utiliza el archivo funciones_habitos.py que posee el codigo de las funciones tanto registrar_datos como analizar_datos. Aqui tambien se pueden ver los docstrings de cada funcion. 
Luego, posee un archivo que posee el codigo del porgrama principal, que utiliza las funciones del archivo anteior, llamado programa_principal.py
